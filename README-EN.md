### ★ YouTube Script for simpleTV
<details>
---

📌 **Installation**
- put **`YT.lua`** in _`..\luaScr\user\video`_


📌 **Search from the "Open URL" window _Ctrl+N_**

prefix | name
------------ | -------------
\- | video
-- | playlist
--- | channel
-+ | live

📌 **Show playlist / select quality: _Ctrl+M_**
